07 · Proteção da comunidade
Todos os dados são coletados com consentimento explícito, conforme a LGPD. O Rossandro é tratado como controlador — decide como os dados são usados e os leva consigo se a parceria terminar. A CAPIO opera como operadora — processa os dados exclusivamente para gerar a experiência, sem utilizá-los para qualquer outra finalidade. Dados sobre estado emocional e jornadas de cuidado recebem proteção adicional, reconhecendo sua natureza sensível.
